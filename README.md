# Oasis Launcher — distribution

Public host for the Oasis game launcher (server source is private, elsewhere).

- `version.json` — launcher self-update info
- `updates.json` — the in-launcher "Recent Updates" feed (edit this to post news)
- `client-manifest.json` — which client the launcher downloads + its SHA-256

The client exe and `OasisLauncherSetup.exe` are attached to this repo's **Releases**.
